### SimpleMeteo

We build fast, privacy-first weather tools — free to use, no accounts needed.

**Our apps:**

- ☀️ [UVI Today](https://uvi.today) — UV index forecast with WHO protection guidelines. Satellite data from Copernicus CAMS. 10 languages.
- 🌿 [Pollen Today](https://pollen.today) — Pollen & allergen forecasts for allergy sufferers.

**How we're different:**

- No tracking, no cookies, no personal data collection
- All weather data from open governmental sources (Copernicus, ECMWF, Open-Meteo)
- Built with Next.js, open APIs, served via Cloudflare
